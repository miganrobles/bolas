1º El usuario introduce el numero de bolas que se van a crear.
2º Se crearan tantas bolas como el usuario haya indicado.
3º La bolas tendrán un color aleatorio.
4º Se crean las bolas con el tamaño de forma aleatoria.
5º La posicion de inicio tambien sera aleatoria en la parte izquierda de la pantalla.
6º Cuando alguna bola llegue a la posicion que se sale del suelo finalizar la animación.
